Extract French company data from the INSEE SIRENE API and push it into a Notion database. Use this skill whenever the user wants to find, list, or extract French companies by department, city, employee count (tranche d'effectifs), or NAF/APE code and populate a Notion database with the results. Also triggers when the user mentions SIRENE, INSEE, annuaire des entreprises, base SIRENE, recherche d'entreprises françaises, or wants to build a prospect list of French companies. Works for any combination of geographic and size filters.
